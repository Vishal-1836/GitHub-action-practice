🟢 BASIC LEVEL (Foundation Concepts)

Problem 1 — Your First Workflow (Triggers & Events)
Concept Covered: Workflow structure, on triggers, jobs, steps, runs-on
Scenario:
You just joined a team and need to create your first CI workflow.
Tasks:

Create a workflow named hello-ci.yml that triggers on:

Push to main branch
Pull request to main branch
Manual trigger (workflow_dispatch)


Print the following in logs:
  Event name that triggered the workflow
  
  1. Branch name
  
  2. Commit SHA
  
  3. GitHub actor (who triggered it)
  
  4. Run on ubuntu-latest
  
  Success Criteria: Workflow runs on all 3 trigger types and shows correct context info.
