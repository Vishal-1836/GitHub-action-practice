Problem 3 — Secrets & Variables Management

Concept Covered: Repository secrets, environment secrets, repository variables, masked logs

Scenario:

You need to connect to a database in CI without exposing credentials.

Tasks:

Create 3 repository secrets:

DB_PASSWORD
API_KEY
AZURE_CLIENT_ID


Create 2 repository variables:
DB_HOST
DB_NAME


Build a workflow that:

Reads all secrets and variables
Tries to print DB_PASSWORD directly (observe masking ***)
Validates secret existence (fail if missing)
Uses secret in an environment variable for a step

Success Criteria: Secrets appear masked in logs; workflow fails gracefully when secret is missing.
