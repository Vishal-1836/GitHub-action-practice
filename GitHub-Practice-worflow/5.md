**Problem 5 — Matrix Strategy for Cross-Platform Testing**

Concept Covered: strategy.matrix, fail-fast, include, exclude, parallel execution

Scenario:

Your library must work across multiple OS and Node.js versions.

Tasks:

Test on this matrix:


OS: ubuntu-latest, windows-latest, macos-latest

Node: 18, 20, 22

Exclude the combination: windows-latest + Node 18

Include an extra entry: ubuntu-latest + Node 21 + experimental: true

Set fail-fast: false so all combinations run even if one fails

For experimental jobs, use continue-on-error: true
