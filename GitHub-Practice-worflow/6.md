**Problem 6 — Caching Dependencies for Faster Builds**

Concept Covered: actions/cache, cache keys, restore keys, cache invalidation

Scenario:

Your CI takes 8 minutes because npm install runs every time. You need to bring it under 2 minutes.

Tasks:

  Cache ~/.npm directory
  Cache key based on package-lock.json hash:
  key: ${{ runner.os }}-node-${{ hashFiles('**/package-lock.json') }}


  Add a fallback restore-keys:
  ${{ runner.os }}-node-


  Verify cache hit/miss in logs
  Compare build times: first run vs cached run

Success Criteria: Second run shows "Cache restored successfully" and finishes 3x faster.
